### How to use
1. Clone this respository
```
    git clone https://github.com/brandonsousa/RandomDice.git
```
2. In the folder
```
    npm install
```
3. If you have the [Expo-cli](https://expo.io/tools#cli)
```
    expo start
```
    - else
    ```
        npm start
    ```
